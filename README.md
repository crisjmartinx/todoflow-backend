# TodoFLOW API

1. Clone proyect.
2. `yarn install`.
3. Clone file `.env.template` rename to `.end`.
4. Change environments variables.
5. Build database.

```
docker-compose up -d
```

6. Run: `yarn start:dev`

7. Run SEED.

```
http://localhost:3000/api/seed
```
